Multi-Source News Dashboard

Comprehensive Web Scraping & Analytics Platform

Project Overview

Real-time visualization dashboard for news content analysis from four major news sources. Collects and processes data from ABC News, CBS News, The Tab, and BuzzFeed to provide insights into digital engagement patterns and content trends.

Tech Stack

Python BeautifulSoup Streamlit PostgreSQL AWS Aurora SQLAlchemy
READ MORE

Problem Statement

Research teams needed a comprehensive system to collect and analyze news content from multiple sources for digital engagement studies. Manual data collection was time-consuming and inconsistent, while existing tools couldn't handle the scale and diversity of news sources required.

Solution Approach

Engineered a comprehensive web scraping system using Python and BeautifulSoup to extract news articles from four major news sources. Built a real-time visualization dashboard using Streamlit and Plotly with automated data refresh and interactive analytics, with all data stored in AWS Aurora PostgreSQL database.

Key Features

  • Multi-source news scraping (CBS, ABC, BuzzFeed, The Tab)
  • Real-time interactive dashboard with 6+ visualization charts
  • Automated data refresh every 10 minutes
  • Automated AWS storage and processing
  • Interactive filtering by source, date, and keywords

Technical Implementation

  • BeautifulSoup for HTML parsing and data extraction
  • Streamlit for interactive web application
  • Plotly for advanced data visualizations
  • PostgreSQL on AWS Aurora RDS for data storage
  • Environment-based configuration for security

Challenges & Solutions

Challenge: Data Integration

Multiple news sources had different data structures and formats that needed standardization.

Solution:

Implemented comprehensive data cleaning and column mapping to standardize data from all sources into a unified format for consistent analysis.

Challenge: Real-time Performance

Dashboard needed to handle large datasets while maintaining fast loading times and interactive features.

Solution:

Built efficient database queries with Streamlit caching (10-minute refresh), optimized data processing, and interactive filtering for smooth user experience.

Challenge: Deployment & Security

Secure deployment required proper credential management and scalable architecture.

Solution:

Implemented environment-based configuration, clean separation of concerns, and deployment-ready structure with proper error handling and connection pooling.
